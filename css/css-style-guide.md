### 30 CSS-selectors, about which is useful to remember

1. The symbol asterisk selects all elements on the page.

```css
* {
 margin: 0;
 padding: 0;
}
```

Also, the * symbol can be used for the children of the object.

```css
#container * {
 border: 1px solid black;
}
```

#### Compatibility:

* IE6+
* Firefox
* Chrome
* Safari
* Opera
